    Quick Reference

    How to use RadAssist and radiology fundamentals

    Image Analysis

    Upload or capture an X-ray image and let GPT-4o Vision analyze it. You'll receive a structured report with findings, impression, and follow-up recommendations. Use "Edit in Report Builder" to refine the generated report.

    Report Builder

    1. Select the modality and body part
    2. Enter the clinical history
    3. Optionally upload an X-ray image
    4. Add your findings one by one (use keyword suggestions)
    5. Click "Generate Report" for a structured RSNA-style report
    6. Edit the generated report directly, then copy or save as draft

    Chat

    Use the chat tab to ask any radiology question or look up specific terms. Short terms (e.g., "pneumothorax") trigger a quick lookup. Full questions get a detailed response with keywords, findings, and differentials.

    Report Structure (RSNA Standard)

    • EXAM: Modality and body part
    • CLINICAL INDICATION: Why the study was ordered
    • TECHNIQUE: Imaging protocol used
    • COMPARISON: Prior studies (if any)
    • FINDINGS: Detailed anatomical description
    • IMPRESSION: Summary of key findings
